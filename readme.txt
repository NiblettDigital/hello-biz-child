Hello Biz Child Theme

== Description ==
Hello Biz Child is a child theme for the Hello Biz WordPress theme. This theme allows you to modify the Hello Biz theme while maintaining the ability to update the parent theme.

== Installation ==
1. In your admin panel, go to Appearance > Themes and click the 'Add New' button.
2. Upload the hello-biz-child.zip file
3. Click on the 'Activate' button to use your new theme right away.
